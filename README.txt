🌐 INTERNET MONITOR - Stop Windows from Eating Your Data!

One-click app that shows what's using your internet and blocks Windows from wasting it!


📥 QUICK START

1. Download InternetMonitor.exe
2. Right-click → Run as Administrator (IMPORTANT!)
3. App opens - you're monitoring!

No installation needed! No Python required!


🎯 WHAT THIS DOES

Windows secretly eats your internet! This app:
- Shows EVERY process using your internet (Red = Microsoft, Green = Your apps)
- Warns you when Microsoft uses more than 100 MB
- Blocks ALL Microsoft with ONE CLICK (updates, store, OneDrive, Xbox, telemetry)
- Keeps your apps working - Chrome, games, Discord still work!


🖥️ HOW TO USE

First Time:
1. Right-click EXE → Run as Administrator
2. See the list - Red = Microsoft data thieves!
3. Set your limit (100 MB recommended)
4. Minimize to tray - runs in background

Buttons:
- 🛑 BLOCK - Stops all Microsoft internet instantly
- ➖ Minimize - Hides to system tray (near clock)
- 🔄 Reset - Clears usage counter
- ⏸️ Pause - Stops monitoring temporarily

When Limit Reached (popup asks):
- YES = Block Microsoft now
- NO = Allow 10 more minutes
- CANCEL = Reset counter

Minimize & Restore:
- Minimize hides to tray (near clock)
- Double-click tray icon to restore
- Right-click tray icon for menu

❓ FAQ

Q: Does this block my internet completely?
A: NO! Only Microsoft. Chrome, games, apps all work.

Q: Can I still update Windows?
A: Yes! Click UNBLOCK, update, then BLOCK again.

Q: Is it safe?
A: Yes! Only uses Windows Firewall, easily reversible.

Q: Why Run as Administrator?
A: Required for firewall blocking. Monitoring works without it.

Q: How to restore window from tray?
A: Double-click the icon near clock, or right-click → Show Window.

Q: Does it work on Windows 11?
A: Yes! Works on Windows 10 and 11.


🔧 TROUBLESHOOTING

App won't open:
- Right-click → Run as Administrator
- Check antivirus isn't blocking
- Try running from a short folder path (C:\Monitor\)

Blocking doesn't work:
- Must run as Administrator
- Windows Firewall must be enabled

Tray icon missing:
- Click ^ arrow near clock
- Restart the app
